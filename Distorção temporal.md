# Estudo geral
Dentro do campo arcano, a distorção temporal sempre representou uma barreira e um ideal para pesquisadores, naturalmente visando técnicas como Imortalidade, ou rejuvenescimento. Entretanto, os materiais de estudo para técnicas similares, são escassos, e muitas vezes, codificados em maneiras tão complexas que o estudo deles se torna impossível.
