O pactuário foca em fazer contratos com entidades magicas para receber feitiços e habilidades especiais. pactuários funcionam como uma versão mais estável do mago, frequentemente funcionando também como uma classe física magica.

| Nivel | Habilidade adquirida        | Contratos disponíveis |
| ----- | --------------------------- | --------------------- |
| 1     | Conjuração, Contrato arcano | 1                     |
| 2     |                             | 1                     |
| 3     | Contrato arcano             | 2                     |
| 4     |                             | 2                     |
| 5     | Contrato arcano             | 3                     |
| 6     |                             | 3                     |
**Pontos de vida iniciais 14 + Vigor**
*Pontos de vida nos próximos níveis 2+Vigor*
**Mana Inicial 8+Presença**
*Pontos de vida nos próximos níveis 4+Presença*

**Pericias:** Negociação
